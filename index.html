Teman - teman semua, ini adalah web aplikasi weekly versi ke 3, di dalam web ini ada beberapa menu, yaitu :

  1. Weekly
  2. Raport
  3. Berkas

Keterangan :

Adapun Weekly tidak menggunakan gform lagi namun berupa form input, untuk memudahkan pengisian di sertakan tombol edit dan delet untuk kemudahan menginput dan mengedit jika ada kesalahan.

Menu Raport berisi raport narasi dan porto versi excel, mungkin sedikit berbeda tampilan dari hasil karena file asli menggunakan Ms. Word, adapun pengisiannya otomatis dengan hasilnya nanti di download dalam bentuk excel (dalam tahap pengembangan)

Menu Berkas adalah menu penyimpanan online (gdrive) yang bertujuan untuk menyimpan file-file administrasi SM Alam Karawang, untuk memudahkan pencarian file-file kedinasan dan sekolah.

Semoga kedepan dalam administrasi di berikemudahan.
